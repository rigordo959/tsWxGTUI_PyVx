   The TeamSTARS "tsWxGTUI" Toolkit is designed for use by
   developers of Python application programs that run on
   32-/64-bit processors under various releases of Linux,
   Mac OS X, Microsoft Windows and Unix operating systems.

   It is implemented and released in two versions:
 
   1. It's "tsWxGTUI_Py2x" release supports application
      program development in the second generation Python
      language (Python 2.0.0-2.7.8).
 
   2. It's "tsWxGTUI_Py3x" release supports application
      program development in the third generation Python
      language (Python 3.0.0-3.4.2).

   3. From the root directory associated with the release
      ("tsWxGTUI_Py2x" or "tsWxGTUI_Py3x"):

      a. Equipment Operator Document(s), in Plain Text
         Format, are to be found in the subdirectory named
         "./tsDocCLI".

         The subdirectory named "./tsDocCLI/UsageTermsAnd
         Conditions" contains the following files:

            "$README-UsageTermsAndConditions.txt"
            "COPYRIGHT.txt"
            "LICENSE.txt"
            "NOTICES.txt"
            "SplashScreenDesignersGuide.txt"

      b. Engineering Document(s), in Adobe PDF and Micro-
         soft Office Formats, are to be found in the sub-
         directory named "./tsDocGUI".

         The subdirectory named "./tsDocCLI/Adobe-PDF-
         Files/" contains the following file:

            "tsWxGTUI_Vol.__3_SDIST_Terms_&_Conditions.pdf"

         The file contains the following topics:

            "$README-UsageTermsAndConditions.txt"
            "COPYRIGHT.txt"
            "LICENSE.txt"
            "NOTICES.txt"
            "SplashScreenDesignersGuide.txt"
