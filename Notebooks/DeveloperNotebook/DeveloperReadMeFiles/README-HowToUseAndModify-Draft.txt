   +----+----+         Command Line Interface (CLI)
   | G T U I |      and "Curses"-based "wxPython"-style, 
   +---------+         Graphical-Text User Interface (GUI)

   on character-mode 8-/16-color (xterm-family) & non-color

   Toolkit subdirectory named "./tsWxGTUI_PyVx/Documents".
