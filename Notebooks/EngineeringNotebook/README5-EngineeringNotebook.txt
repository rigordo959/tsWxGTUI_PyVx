   This directory contains a collection of files, in appli-
   cation-specific formats, which provide the Toolkit archi-
   tect, product manager, project engineer, system engineer,
   software engineer, test engineer and troubleshooter with:

   1.1 Product Marketing Documentation

       a) announcement notice(s)
       b) brochure(s)
       c) introduction
       d) slide-show presentation

   1.2 Project Documentation

       a) goal (capability) objectives
       b) non-goal (limitation) constraints
       c) project plans

   1.3 Technical Documentation

       a) architectural plans
       b) system specifications
       c) interface specifications
       d) software requirement specifications
       e) design specifications
       f) test specifications
       g) software user manual
